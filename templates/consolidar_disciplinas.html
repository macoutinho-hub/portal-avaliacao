Consolidar Nomes de Disciplinas

Algumas pautas usam abreviaturas ou grafias diferentes para a mesma disciplina (ex.: ECO. C / Econ C em vez de Economia C, ou MAT.A / Mat A em vez de Matemática A), o que faz com que apareçam como colunas separadas na ficha do aluno.

Ferramenta manual — escolha você as correspondências, uma de cada vez. Para cada nome de disciplina existente, selecione no menu para qual nome ele deve ser fundido e prima "Fundir". Quando exista uma sugestão automática plausível, ela já vem pré-seleccionada — mas pode sempre escolher outra opção ou deixar em branco para não fazer nada.
  • Fundir renomeia todos os registos do nome de origem para o nome de destino;
  • quando já existe um registo com o nome de destino para o mesmo aluno (e período / ano lectivo), o registo de origem é removido em vez de renomeado, para não duplicar notas;
  • cuidado com disciplinas parecidas mas distintas — por exemplo Inglês e Líng. Estrang. I - Inglês, ou Filosofia e Filosofia A, não devem ser fundidas.
